📝 ตัวอย่าง config.json:
─────────────────────────────────────────────────────────

{
  "mt5": {
    "path": "C:/Program Files/MetaTrader 5/terminal64.exe",
    "login": 12345678,
    "password": "your_password",
    "server": "Exness-MT5Real",
    "auto_connect": true          ← เปิดนี้!
  },
  "chart": {
    "symbol": "XAUUSD",           ← Symbol ที่ต้องการ
    "timeframe": "H1",
    "bars": 2000,
    "auto_load": true             ← เปิดนี้!
  },
  "pnf": {
    "box_size": 100,
    "reversal_amount": 3
  }
}


🚀 วิธีอัปเดต:
─────────────────────────────────────────────────────────

1. ดาวน์โหลด pnf_mt5_project.zip (v1.1.0)
2. แตกไฟล์
3. แก้ไข config.json:
   • ใส่ MT5 login, password, server
   • ใส่ symbol ที่ต้องการ
   • ตั้ง auto_connect = true
   • ตั้ง auto_load = true
4. รัน: python main.py
5. เปิด: http://localhost:8000
6. ✨ เสร็จ! ระบบทำงานอัตโนมัติ

⚙️ การตั้งค่าที่แนะนำ:
─────────────────────────────────────────────────────────

สำหรับ Production:
  "auto_connect": true
  "auto_load": true
  → ใช้งานได้ทันที

สำหรับ Testing:
  "auto_connect": false
  "auto_load": false
  → ควบคุมด้วยตัวเอง
