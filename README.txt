⚡ วิธีติดตั้งด่วน:
────────────────────────────────────────────────────────

สำหรับผู้ใช้เก่า:
1. สำรอง config.json
2. แทนที่ static/index.html ด้วยไฟล์ใหม่
3. Refresh browser (Ctrl+F5)
4. เสร็จ! ทดสอบ Zoom และ Pan

4. เสร็จ! พร้อมเทรด

🎯 ฟีเจอร์เด่น:
────────────────────────────────────────────────────────

Pan & Zoom:
  • Zoom 50% ถึง 500%
  • เลื่อนด้วยลูกกลิ้งเมาส์
  • ลากเมาส์เพื่อ Pan
  • Pinch-to-Zoom บนมือถือ

Price Labels:
  • ด้านซ้าย: ราคาหลัก (ทุก 5 แถว)
  • ด้านขวา: ราคาทุกแถว! (ใหม่)
  • สีสลับกันเพื่อความชัดเจน

Auto-Refresh:
  • ตั้งค่า 5-300 วินาที
  • Toggle ON/OFF ได้
  • Silent Refresh
  • แสดงเวลา Update

Auto-Connect:
  • เชื่อมต่อ MT5 อัตโนมัติ
  • โหลดกราฟอัตโนมัติ
  • ไม่ต้องกรอกข้อมูลซ้ำ


💡 ตัวอย่างการใช้งาน:
────────────────────────────────────────────────────────

Desktop:
  1. เลื่อนลูกกลิ้งเมาส์ = Zoom
  2. ลากเมาส์ = Pan
  3. กดปุ่ม ⟲ = Reset

Mobile:
  1. Pinch-to-Zoom
  2. ลาก 1 นิ้ว = Pan
