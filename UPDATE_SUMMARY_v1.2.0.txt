🚀 ฟีเจอร์ใหม่ที่ได้:
═══════════════════════════════════════════════════════════

1. 🎮 Pan & Zoom แบบ TradingView
   ────────────────────────────────────────────────────────
   • Zoom In/Out: ลูกกลิ้งเมาส์ หรือปุ่ม +/−
   • Pan: ลากเมาส์เพื่อเลื่อนดู
   • Mobile: Pinch-to-Zoom และ Drag
   • Zoom Range: 50% ถึง 500%
   • Reset Zoom: กดปุ่ม ⟲

2. 📊 แสดงราคาทุกแถว
   ────────────────────────────────────────────────────────
   • ด้านซ้าย: ราคาหลักทุก 5 แถว (ใหญ่)
   • ด้านขวา: ราคาทุกแถว! (ใหม่)
   • สีสลับกันเพื่อความชัดเจน
   • อ่านง่ายแม้ Zoom

3. 🔄 Auto-Refresh
   ────────────────────────────────────────────────────────
   • รีเฟรชข้อมูลอัตโนมัติ
   • ตั้งค่า 5-300 วินาที
   • Silent Refresh (ไม่มี Loading)
   • เก็บตำแหน่ง Zoom/Pan ไว้
   • แสดงเวลา Update ล่าสุด

4. ⚡ Auto-Connect (จาก v1.1.0)
   ────────────────────────────────────────────────────────
   • เชื่อมต่อ MT5 อัตโนมัติ
   • โหลดกราฟอัตโนมัติ
   • ไม่ต้องกรอกข้อมูลซ้ำ

📥 วิธีติดตั้ง/อัพเดท:
═══════════════════════════════════════════════════════════

สำหรับผู้ใช้เก่า (มี v1.0.x หรือ v1.1.0):
────────────────────────────────────────────────────────

1. สำรองไฟล์สำคัญ:
   ✓ สำรอง config.json (มี password)

2. แทนที่ไฟล์:
   ✓ แทนที่ static/index.html ด้วยไฟล์ใหม่

3. Restore config:
   ✓ ใช้ config.json เดิม (ไม่ต้องแก้)

4. Test:
   ✓ Refresh browser: Ctrl+F5 (Windows) หรือ Cmd+Shift+R (Mac)
   ✓ ทดสอบ Zoom: เลื่อนลูกกลิ้งเมาส์
   ✓ ทดสอบ Pan: ลากเมาส์
   ✓ ทดสอบ Auto-Refresh: กดปุ่ม Toggle


สำหรับผู้ใช้ใหม่:
────────────────────────────────────────────────────────

1. ติดตั้ง Dependencies:
   > pip install -r requirements.txt

2. แก้ไข config.json:
   • ใส่ MT5 login, password, server
   • ตั้ง auto_connect = true
   • ตั้ง auto_load = true

3. รันโปรแกรม:
   > python main.py

4. เปิด Browser:
   http://localhost:8000

  }

🎯 การใช้งานอย่างรวดเร็ว:
═══════════════════════════════════════════════════════════

Desktop:
────────────────────────────────────────────────────────
1. Zoom:
   • เลื่อนลูกกลิ้งเมาส์ขึ้น/ลง
   • หรือกดปุ่ม + / −

2. Pan:
   • คลิกค้างเมาส์ซ้าย แล้วลาก

3. Reset:
   • กดปุ่ม ⟲ หรือ "Reset Zoom"


Mobile:
────────────────────────────────────────────────────────
1. Zoom:
   • ใช้ 2 นิ้ว Pinch-to-Zoom

2. Pan:
   • ลาก 1 นิ้ว


Auto-Refresh:
────────────────────────────────────────────────────────
1. ตั้งค่า Interval (แนะนำ 30 วินาที)
2. กดปุ่ม "🔄 Toggle Auto-Refresh"
3. ดู Indicator เป็นสีเขียว = เปิดแล้ว
